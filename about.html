The GNU Bourne Again SHell is the default shell on Linux and Apple MacOS X. It has a wealth of features, many of which are hidden deep in documentation. On this site you will find some tips and tricks for using Bash.

While you may not find the most detailed of explanations here, you can always check the man pages of your shell or the bash reference manual online for further information.

If you find a typo, have suggestions for improvements or want to submit a post, feel free to create a fork on github and send a pull request. You'll need Ruby, Jekyll and Sass to build the site.

The icon used in the logo and as the favicon is from the silk icon set from famfamfam.com.
